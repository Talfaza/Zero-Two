## Environment Setup :

- Linux (RHEL) :

```bash
$ sudo dnf install gcc-c++ make cmake 
```
## Setup : 

```bash
git clone https://github.com/Talfaza/Zero-Two/
cd Zero-Two/compiler
make clean;make
./zerotwo <file>.zt
```
